üöÄ Getting Started

Tools: Compiler (g++, clang) and an editor (VS Code recommended). Below are quick commands for compiling and running on the command line.

// Save as hello.cpp
#include <iostream>
using namespace std;

int main(){
  cout << "Hello, World!\n";
  return 0;
}

Compile & run (Linux / macOS / WSL):

g++ hello.cpp -o hello
./hello

Compile & run (Windows using MinGW):

g++ hello.cpp -o hello.exe
hello.exe

üîÅ Object-Oriented Programming

Classes, objects and basic encapsulation.

class Rectangle {
public:
  Rectangle(int w, int h): width(w), height(h){}
  int area(){ return width * height; }
private:
  int width;
  int height;
};

// usage
Rectangle r(3,4);
cout << r.area() << '\n';

üìù Exercises

Exercise 1 ‚Äî Hello Input:

Read a name from input and print Hello, <name>!

#include <iostream>
#include <string>
using namespace std;
int main(){
  string name;
  getline(cin, name);
  cout << "Hello, " << name << "!\n";
  return 0;
}
Exercise 2 ‚Äî Sum of N:

Read N then N integers, print the sum.

#include <iostream>
using namespace std;
int main(){
  int N; cin >> N;
  long long s = 0;
  for (int i=0;i<N;++i){ int x; cin >> x; s += x; }
  cout << s << '\n';
}
